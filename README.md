# SOCIAL INSTA - গেমিং অ্যাপ

একটি সুন্দর এবং ইন্টারঅ্যাক্টিভ সোশ্যাল মিডিয়া গেমিং অ্যাপ্লিকেশন।

## 🎮 ফিচার

- **মাল্টি-ট্যাব নেভিগেশন**: হোম, গেম, অ্যাচিভমেন্টস, প্রোফাইল
- **গেম মেকানিক্স**: লেভেল প্রগ্রেশন, ডায়মন্ড কারেন্সি, স্পিন হুইল
- **প্রোফাইল সিস্টেম**: নাম, বায়ো, ভিউ, ডায়মন্ড ব্যালেন্স এডিট
- **ভিআইপি মেম্বারশিপ**: বিশেষ সুবিধা এবং ডাবল বোনাস
- **অ্যাচিভমেন্টস**: পুরস্কার এবং ট্রফি সিস্টেম
- **মাল্টিলিঙ্গুয়াল**: বাংলা, ইংরেজি, হিন্দি সাপোর্ট
- **ডার্ক/লাইট মোড**: থিম সিলেক্টর
- **ডেটা পার্সিস্টেন্স**: localStorage এ সব ডেটা সংরক্ষিত

## 🚀 শুরু করুন

### প্রি-রিকোয়ারমেন্ট
- Node.js (v14 বা তার উপরে)
- npm বা yarn

### ইনস্টলেশন

```bash
# রিপোজিটরি ক্লোন করুন
git clone https://github.com/bbiswanath464-netizen/Social-insta-.git
cd Social-insta-

# ডিপেন্ডেন্সি ইনস্টল করুন
npm install

# ডেভেলপমেন্ট সার্ভার চালান
npm run dev
```

ব্রাউজার খুলুন এবং `http://localhost:5173` এ যান।

## 📦 বিল্ড

প্রোডাকশনের জন্য বিল্ড করুন:

```bash
npm run build
```

এটি `dist` ফোল্ডারে অপটিমাইজড ফাইল তৈরি করবে।

## 🛠️ প্রযুক্তি

- **React** - ইউজার ইন্টারফেস
- **Vite** - বিল্ড টুল এবং ডেভেলপমেন্ট সার্ভার
- **Tailwind CSS** - স্টাইলিং
- **Lucide React** - আইকন

## 📁 ফোল্ডার স্ট্রাকচার

```
Social-insta-/
├── src/
│   ├── App.jsx        # মেইন অ্যাপ কম্পোনেন্ট
│   ├── main.jsx       # এন্ট্রি পয়েন্ট
│   └── index.css      # গ্লোবাল স্টাইলস
├── index.html         # HTML টেমপ্লেট
├── package.json       # ডিপেন্ডেন্সি এবং স্ক্রিপ্ট
├── vite.config.js     # Vite কনফিগারেশন
├── tailwind.config.js # Tailwind কনফিগারেশন
├── postcss.config.js  # PostCSS কনফিগারেশন
└── README.md          # এই ফাইল
```
